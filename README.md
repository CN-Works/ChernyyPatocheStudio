## Translation

## Dependency
*- RageUix by @Florian-1801 (already installed in "src" folder)*

*- ox_lib (https://github.com/overextended/ox_lib)*

## Installation
*- Drag and drop the folder into your resource folder*

*- start the resource in your server.cfg !*
